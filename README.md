# AI Attendance System using OpenCV

## Overview

A Computer Vision based Attendance Management System developed using Python and OpenCV.

The system detects faces through webcam input and records attendance automatically with timestamps and image capture.

---

## Features

- Real-time webcam access
- Face detection using Haar Cascade
- Attendance logging with date and time
- CSV file storage
- Automatic image capture
- Duplicate attendance prevention
- Student name-based attendance tracking
- Professional OpenCV user interface

---

## Technologies Used

- Python
- OpenCV
- NumPy
- CSV File Handling
