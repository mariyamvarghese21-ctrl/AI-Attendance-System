# AI Attendance System using OpenCV

## Overview
A Computer Vision–based attendance management system developed using Python and OpenCV. The system detects faces through webcam input and records attendance automatically with timestamps and image capture.

## Features
- Real-time webcam access
- Face detection using Haar Cascade
- Attendance logging with date and time
- CSV file storage
- Automatic image capture
- Duplicate attendance prevention
- Student name-based attendance tracking

## Technologies Used
- Python
- OpenCV
- NumPy
- CSV File Handling

## How to Run

1. Install required libraries:

```bash
pip install -r requirements.txt
```

2. Run the program:

```bash
python main.py
```

3. Press:
- A → Mark Attendance
- Q → Quit
